# SNSProject
個人開発のSNSです。フロントエンドをVue.js、バックエンドをDjango REST framework、それらをDockerで構築しています。

## 本サービスの特徴
投稿された内容が投稿日時に関わらずランダムに表示されます。バルーンリリース（風船に手紙を付けて飛ばすイベント）を意識して開発しました。
